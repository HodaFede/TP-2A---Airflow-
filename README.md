# TP-2A---Airflow-
L'objectif de ce TP est de découvrir les bases d'Apache Airflow en concevant un pipeline d'ingestion de données météo. Il s'agit d'apprendre à découper un traitement en étapes métier distinctes (extraction, transformation et chargement) afin de construire un workflow clair, maintenable et conforme aux bonnes pratiques du Data Engineering.
